# OpenFindBearings Web

Vue 3 + TypeScript + Vite 前端应用，OpenFindBearings 的 Web 端实现。

> 未来计划通过 Flutter / React Native 扩展为移动端应用。

## 快速开始

```bash
npm install
npm run dev
```

## 常用命令

```bash
npm run dev       # 启动开发服务器
npm run build     # 生产构建
npm run preview   # 预览生产构建
npm run type-check  # TypeScript 类型检查
```
